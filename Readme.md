#大学の同窓のSNSサービス(Team Project)

##サービス
 * 自動的に食事する人をマッチする
 * リアルタイムチャット
 * リアルタイムアラーム
 * フォロー機能
 * 書き込みとコメント機能

 * Node.js 4.3.0
